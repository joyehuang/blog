# Joye's Blog

这是我的个人博客，使用 Hugo 构建，基于 Hugoplate 主题开发。

## 🚀 特点

- 🌏 支持中英文双语
- 🎨 响应式设计
- 🌙 深色模式
- 📝 Markdown 写作
- 🔍 全站搜索
- 📊 Vercel Analytics 集成
- 📨 邮件订阅功能

## 🛠️ 技术栈

- [Hugo](https://gohugo.io/) - 静态网站生成器
- [Tailwind CSS](https://tailwindcss.com/) - 样式框架
- [Vercel](https://vercel.com/) - 部署平台
- [Mailchimp](https://mailchimp.com/) - 邮件订阅服务
