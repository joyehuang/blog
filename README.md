## 本地开发

环境要求：
- [Node.js](https://nodejs.org/): 18.0.0+
- [Bun](https://bun.sh/) (推荐) 或 npm

克隆仓库：

```shell
git clone https://github.com/joyehuang/blog.git
cd blog
```

常用命令：

```shell
# 安装依赖
bun install

# 启动开发服务器
bun dev

# 构建项目
bun run build

# 预览（构建后）
bun preview

# 创建新文章
bun new

# 格式化代码
bun format

# 代码检查
bun lint

# 一键四连（lint + sync + check + format）
bun yijiansilian
```
